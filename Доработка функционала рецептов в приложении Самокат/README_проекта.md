# Доработка функционала рецептов в приложении Самокат

## Контекст

База рецептов блюд в приложении Самокат была архивирована из-за низкой бизнес-ценности и отсутствия значимого влияния на удержание пользователей. Требовалось переосмыслить сценарии использования рецептов и превратить их в инструмент повышения вовлеченности и удержания клиентов.

## Цель проекта

Спроектировать новые пользовательские сценарии использования базы рецептов, повышающие вовлеченность и удержание клиентов за счет упрощения пути от выбора блюда до оформления заказа.

## Артефакты

- Техническое задание (ТЗ)
- BPMN-модель процесса TO-BE
- Интерактивный прототип решения
- Презентация концепции решения

## Результат

Спроектированы новые сценарии использования рецептов в приложении Самокат. Пользователь получил возможность добавлять все ингредиенты выбранного блюда в корзину одной кнопкой, сохранять рецепты и повторно оформлять заказ без повторного поиска товаров.
