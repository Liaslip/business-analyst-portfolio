# Оптимизация витрины мобильного банка ОТП

## Контекст

С развитием мобильного банка продуктовый интерфейс стал перегруженным: большое количество функций и предложений усложняло навигацию, создавало барьеры в клиентском пути и снижало заметность банковских продуктов.

## Цель проекта

Выявить ключевые проблемы пользовательского опыта, проанализировать клиентские сценарии и спроектировать обновленную концепцию интерфейса для упрощения взаимодействия с банковскими продуктами.

## Артефакты

* Customer Journey Map (CJM)
* BPMN-модель процессов AS-IS / TO-BE
* User Story Map
* Интерактивный прототип решения
* Продуктовые гипотезы

## Результат

Выявлены ключевые барьеры клиентского пути, включая перегруженность интерфейса и сложности при выборе продуктов. Спроектирована новая концепция интерфейса с упрощенной навигацией, приоритизацией ключевых пользовательских сценариев и персонализированными продуктовыми предложениями.
