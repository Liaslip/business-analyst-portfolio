# Business Analyst Portfolio

В этом репозитории собраны мои проекты по бизнес-аналитике: кейсы, аналитические артефакты и проектная документация, демонстрирующие мой подход к анализу требований, моделированию процессов и проектированию решений для бизнес-задач.

## Что представлено в репозитории

Внутри каждого проекта представлены:

* Контекст и цель проекта
* Аналитические артефакты 
* Техническая и проектная документация
* Прототипы и визуальные материалы
* Описание результатов
