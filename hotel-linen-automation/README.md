# Автоматизация логистики и учета белья в сети отелей

## Контекст 

С ростом сети отелей процесс учета и перемещения белья между отелями и централизованной прачечной стал неэффективным: отсутствие прозрачного контроля приводило к потерям белья, ошибкам в поставках и задержкам доставки.

## Цель проекта

Выявить причины операционной неэффективности в процессе перемещения белья между отелями сети и централизованной прачечной, а также спроектировать решение для автоматизации учета и логистики.

## Артефакты

* Отчет о предпроектном обследовании
* Техническое задание (ТЗ)
* UML-диаграммы классов и объектов
* UML-диаграмма прецедентов (Use Case Diagram)
* Use Cases и требования к решению
* Программа и методика испытаний (ПМИ)

## Результат

Выявлены ключевые причины операционного хаоса в логистическом процессе и спроектировано решение, обеспечивающее централизованный контроль учета и перемещения белья между всеми участниками процесса.
